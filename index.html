Merge intervals

To merge intervals, we need to identify which intervals overlap or are adjacent to each other. The key idea is to sort the intervals based on their start times. Once the intervals are sorted, we iterate through the collection and compare each interval's start time with the previous interval's end time.

A simple approach is to start from the first interval and compare it with all other intervals for overlapping, if it overlaps with any other interval, then remove the other interval from the list and merge the other into the first interval. Repeat the same steps for the remaining intervals after the first.
